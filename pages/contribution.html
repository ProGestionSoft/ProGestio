CONTRIBUER AU DEVELOPPEMENT

Merci de votre intérêt pour contribuer à PGS Game Bot !
Voici quelques instructions pour vous aider à commencer.

Comment Contribuer

  1. Fork le dépôt : Commencez par forker le dépôt sur GitHub en cliquant sur le bouton Fork en haut à droite de la page du projet.
  2. Clonez votre Fork : Clonez le dépôt forké sur votre machine locale : 
    
            git clone https://github.com/ProGestionSoft/pgsgamebot.git
  3. Créez une branche : Créez une nouvelle branche pour votre fonctionnalité ou correction : 
    
            git checkout -b pgsgamebot
  4. Effectuez les modifications : Apportez les modifications nécessaires dans votre branche. Assurez-vous de bien tester votre code avant de soumettre vos changements.
  5. Committez vos modifications : Ajoutez et commettez vos modifications : 
    
            git add. 
    git commit -m "Description claire de vos modifications"
  6. Poussez votre branche : Poussez votre branche vers votre fork : 
    
            git push origin pgsgamebot
  7. Créez une pull request : Allez sur GitHub et ouvrez une pull request depuis votre branche vers la branche principale (main) du dépôt d'origine. Décrivez clairement les modifications apportées et pourquoi elles sont nécessaires.

Guide de style

  • Respectez les conventions de style de code utilisées dans le projet.
  • Assurez-vous que votre code est bien documenté et commenté.
  • Ajoutez des tests unitaires pour les nouvelles fonctionnalités lorsque cela est possible.

Signalement de Bugs

Si vous rencontrez un bug ou un problème, veuillez ouvrir une issue sur GitHub en fournissant autant de détails que possible pour nous aider à reproduire et résoudre le problème.

Questions

Pour toute question ou clarification, n'hésitez pas à nous contacter via GitHub Discussions ou à nous envoyer un message direct.
